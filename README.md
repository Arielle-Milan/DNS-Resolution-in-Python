# DNS-Resolution-in-Python

## Scenario:
Study the scenario and complete the question(s) that follow(s):

A video streaming platform wants to optimize its DNS resolution process to reduce latency when retrieving video content from different servers. The system must resolve domain names efficiently while implementing caching to minimize redundant queries.
